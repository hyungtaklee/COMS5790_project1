### How to run the project

1. Install UV (require `pip`)
```
python -m install uv
```
or
```
pip install uv
```

2. Run program
```
uv run project1.py
```

## Turn on/off lemmatization
Go to line 17
```
is_lemma = True or False # Apply lemmatization if True.
```
